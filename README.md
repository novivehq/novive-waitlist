# Novive Waitlist Landing Page

This repository contains the official waitlist landing page for **Novive** —  
a digital marketing training platform for aspiring Nigerian marketers.

## What’s included
- Responsive landing page
- Waitlist signup form
- Email integration (Brevo / ConvertKit)
- Thank-you confirmation page

## Tech stack
- HTML
- CSS
- Hosted on Netlify or Vercel

## Deployment
This project can be deployed via:
- Netlify (recommended)
- Vercel
